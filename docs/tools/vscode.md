# vscode 

## 快捷键

### 编辑

| 快捷键                   | 描述            |
| ------------------------ | --------------- |
| Cmd + /                  | 注释/取消注释   |
| Option + Shift + F       | 代码格式化      |
| option + up/down         | 向上/下移动行   |
| Shift + option + up/down | 向上/下复制一行 |
| Ctrl + J                 | 合并代码行      |

### 编辑器
| 快捷键                | 描述                 |
| --------------------- | -------------------- |
| Cmd + up              | 移动开头             |
| Cmd + down            | 移动结尾             |
| F1 或 Cmd + Shift + P | 打开命令面板         |
| Cmd + Shift + O       | 命令面板快速访问符号 |
| Ctrl + G              | 行跳转               |
